GitHub Pages + Custom Subdomain Test

This just a test page. Just wanna try out Github Pages with a custom subdomain from FreeDNS

Quick checks

  • Repo branch: main or gh-pages (depends on your Pages settings)
  • Site URL (GitHub Pages): https://<username>.github.io/<repo>/
  • Custom subdomain: e.g. yourname.freedns.org

If you see your FreeDNS subdomain here (e.g. yourname.freedns.org), DNS is correctly pointed.

GitHub Pages docs

How to

  1. Make an html file and push to your GitHub repo
  2. Enable Pages in repo Settings → Pages (select branch or /docs folder).
  3. Open your subdomain provider and get a subdomain.
  4. When you get your subdomain, use that subdomain to in GitHub pages
  5. Wait a few minutes, then visit your custom subdomain.
